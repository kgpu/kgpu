DynamicStructRef

(jvm) open class DynamicStructRef<T : WgpuJavaStruct!> : PointerField

A rewrite of jnr.ffi.Struct.StructRef to allow chained structs. It achieves this by calculating the size of the struct at the time the memory is used, instead of in the constructor.
